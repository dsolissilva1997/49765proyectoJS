Asientos Avión - Simulador de Reserva

Consiste en un avión con 10 asientos en el cual, mediante un array, se recorre la disponibilidad y se almacena el nombre del pasajero. El estado de los asientos y el carrito se actualiza correctamente. Se utilizan funciones, if y else para comprobaciones y métodos para el array.

En este último proyecto de entrega, se utiliza Fetch con promesas, se añaden promesas a otras funciones de js y se añade un botón para ver el carrito en el HTML
